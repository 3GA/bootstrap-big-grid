Big Grid

The default Bootstrap Grid only provides grid dimensions up to 1200px. Using the same mixins and variables provided by Bootstrap, the grid has been extended to support various larger resolution screens, such as HDTVs and the 15" Retina MacBook Pro.

Please consult the original Bootstrap Grid docs for an introduction to the Bootstrap Grid and its usage.

Grid Options

720p TV ≥1366px 1080p TV ≥1920px 1440p TV ≥2560px 15" Retina MacBook Pro ≥2880px 4K TV ≥4096px 8K TV ≥8192px
Grid behavior Collapsed to start, horizontal above breakpoints
Class prefix .col-tv720p- .col-tv1080p- .col-tv1440p- .col-retina15- .col-tv4k- .col-tv8k-
# of columns 12
Gutter width 30px (15px on each side of a column)
Nestable Yes
Offsets Yes
Column ordering Yes
